## 👨‍💻 About This Project

This project was built as part of my data engineering and analytics portfolio to solve a real-world problem in India’s energy sector.

I focused on:
- Understanding fuel and LPG pricing challenges in India  
- Designing meaningful data models  
- Building an interactive dashboard for insights  
- Simulating real-world analytics workflows  

---

## 🎯 Overview

A comprehensive analytics dashboard inspired by real-world use cases in India’s energy sector. It helps analyze fuel price trends, LPG affordability, and regional disparities affecting millions of households.

- 📊 Real-time fuel price monitoring across states  
- 💰 LPG affordability analysis relative to income  
- 🚨 Alert system for price spikes and anomalies  
- 📰 News sentiment tracking  
- 🎯 Custom metrics like "Energy Stress Index"  

---

## ✨ Key Features

### 📈 National Overview
- KPI cards for Petrol, Diesel, LPG prices  
- Daily and weekly trend indicators  
- 30-day price trend visualization  
- Inflation impact indicator  

### 🗺️ State-wise Analysis
- State comparison charts  
- Price variation insights  
- Regional inequality visualization  

### 💡 Affordability & Insights
- **Energy Stress Index** (LPG cost vs income)  
- Subsidy and affordability trends  
- Identification of high-burden states  

### 🚨 Alerts & Anomalies
- Detection of sudden price spikes  
- Basic anomaly identification  
- Priority-based alert system  

### 📰 News & Sentiment
- News tracking related to fuel/LPG  
- Sentiment classification (positive/negative/neutral)  
- Correlation with price trends

## 📊 Key Metrics

### Energy Stress Index
**Formula**: `(LPG Price / Average Monthly Income) × 100`

**Insight Example**:
Higher percentage indicates higher financial burden on households.
